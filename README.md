# 异常信息填报系统

沙钢永兴特钢技术质量处验收班异常信息填报系统（纯前端版）

## 特性

- 🌐 **纯前端实现** - 无需后端服务器，数据存储在浏览器本地
- 📱 **移动端优先** - 完美适配手机浏览器，支持PWA安装
- 📊 **数据可视化** - Chart.js 图表展示异常趋势和物料分布
- 📥 **导出Excel** - 使用 SheetJS 生成标准 Excel 文件
- 🔄 **数据共享** - 支持 JSON 导入导出，方便汇总数据

## 快速开始

### 方式一：本地直接打开

```bash
# 使用任意静态服务器
python -m http.server 8080
# 然后访问 http://localhost:8080
```

### 方式二：部署到 GitHub Pages

1. 将 `index.html`、`manifest.json`、`sw.js` 推送到 GitHub 仓库
2. 进入仓库 **Settings** → **Pages**
3. Source 选择 `main` 分支
4. 等待部署完成，访问 `https://你的用户名.github.io/仓库名`

## 功能说明

### 填报
- 选择日期（默认当天）
- 选择物料名称（彩色标签）
- 填写供货单位、车号
- 描述异常情况
- 选择两位验收员

### 记录
- 时间线展示所有异常记录
- 支持按全部/本周/本月筛选
- 支持编辑和删除（仅本人或管理员）

### 我的
- 查看个人填报统计
- 导出Excel（按周/月/自定义范围）
- 导出JSON（用于备份和分享）
- 导入JSON（统计员汇总数据用）
- 清空本地数据

## 技术栈

- HTML5 + CSS3 + JavaScript（原生）
- [Chart.js](https://www.chartjs.org/) - 图表可视化
- [SheetJS (xlsx.js)](https://sheetjs.com/) - Excel导出
- localStorage - 本地数据存储
- PWA - 支持离线访问和安装到桌面

## 文件说明

| 文件 | 说明 |
|------|------|
| index.html | 主应用（包含全部HTML/CSS/JS） |
| manifest.json | PWA配置文件 |
| sw.js | Service Worker（离线缓存） |

## 注意事项

1. **数据备份** - 更换设备前请导出JSON备份
2. **数据隔离** - 每台设备数据独立存储
3. **数据汇总** - 统计员通过导入JSON合并各验收员数据
4. **浏览器兼容** - 推荐使用 Chrome、Safari、Edge 最新版
